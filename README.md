# AI Act Compliance Dodah.

This is a quick and dirty repo looking at some tooling to maybe meet the requirements of the EU AI Act (at least those in Chapter 3).

You probably want `compliance_stack.md` -- which shows nicely on github.com. I've suggested using `marp` in VSCode, if you want to use vscode to browse/work with the repo.

The `compliance/` directory has individual files as devised by Gemini -- they should all be contained in the `compliance_stack.md` file though. 

`github_actions.md` is a sample file for setting up GitHub Actions with the full toolchain recommended.

In addition to the AI Act, a few touch on the CRA, as well as requirements of ISO 42k.

Typically, I would have added a markdown, yaml, json linter as hooks, but I've not in this instance out of laziness.
